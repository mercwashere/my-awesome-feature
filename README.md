# Instructions for setting up the game

First, you have to make a python virtual environment and install the required packages:

```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

```

# How to run the game

Make sure your venv is activated (it will be if you followed the instructions above!)
Then, run the game:

```
python main.py
```
